Vercel is the Frontend Cloud. Build, scale, and secure a faster, more personalized web with the frameworks, workflows, and infrastructure your team needs.

Vercel provides the developer tools and cloud infrastructure to build, scale, and secure a faster, more personalized web.

## Key Features

Vercel's platform includes a comprehensive set of features designed for modern web development:

- **Instant Deployments**: Push to Git and get a live URL in seconds. Zero configuration required.
- **Preview Deployments**: Every pull request gets its own live URL for review and testing.
- **Edge Network**: Content is served from 100+ edge locations worldwide for blazing-fast performance.
- **Framework Support**: First-class support for Next.js, Nuxt, SvelteKit, Astro, Remix, and 35+ more.
- **Serverless Functions**: API routes and server-side logic that scale automatically.
- **Edge Functions**: Run code at the edge with sub-millisecond cold starts.
- **Web Analytics**: Real user monitoring and Core Web Vitals tracking.
- **Enterprise Security**: SOC 2 compliance, DDoS protection, and custom firewalls.
