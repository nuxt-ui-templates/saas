## Deploy Your First Project

The fastest way to get started is to import an existing Git repository:

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import your Git repository from GitHub, GitLab, or Bitbucket
3. Vercel detects your framework and configures build settings automatically
4. Click **Deploy** and your project is live

## Using the CLI

You can also deploy from the command line:

```bash [Terminal]
npm i -g vercel
vercel
```

The Vercel CLI will guide you through linking your project, configuring settings, and deploying to production.

## Framework Quick Starts

Start a new project with your favorite framework:

```bash [Next.js]
npx create-next-app@latest my-app
cd my-app
vercel
```

```bash [Nuxt]
npx nuxi init my-app
cd my-app
vercel
```

```bash [SvelteKit]
npx sv create my-app
cd my-app
vercel
```

That's it! Your project is deployed and accessible via a `.vercel.app` URL. Connect a custom domain from the Dashboard at any time.
