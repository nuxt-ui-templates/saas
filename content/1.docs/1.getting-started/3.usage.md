## Project Settings

Configure your project settings from the Vercel Dashboard:

- **Build Command**: Override the auto-detected build command
- **Output Directory**: Specify a custom output directory
- **Install Command**: Customize the dependency installation command
- **Root Directory**: Set a subdirectory as the project root for monorepos

## Environment Variables

Add environment variables from the Dashboard or CLI. Variables can be scoped to specific environments:

- **Production**: Applied to production deployments only
- **Preview**: Applied to preview deployments
- **Development**: Available when running `vercel dev` locally

```bash [Terminal]
vercel env add DATABASE_URL
```

## Custom Domains

Connect custom domains to your project from the Dashboard. Vercel automatically provisions SSL certificates and configures DNS settings. Wildcard domains are supported on Pro and Enterprise plans.
